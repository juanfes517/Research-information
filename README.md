# Como ejecutar

- Crear un entorno virtual:
  ```
  python -m venv env
  ```

- Activar el entorno virtual creado:
  ```
  env\Scripts\activate
  ```

- Instalar las siguientes dependencias:
  ```
  pip install selenium
  ```
  ```
  pip install webdriver-manager
  ```
  ```
  pip install pandas
  ```
