# Como ejecutar

- Crear un entorno virtual:
  ```
  python -m venv env
  ```

- Activar el entorno virtual creado:
  ```
  env\Scripts\activate
  ```

- Instalar selenium:
  ```
  pip install selenium
  ```

- Instalar webdriver-manager:
  ```
  pip install webdriver-manager
  ```

- Instalar pandas:
  ```
  pip install pandas
  ```
